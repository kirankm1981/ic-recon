# IC Recon - Intercompany Reconciliation Platform

  Enterprise-grade intercompany reconciliation platform built with React, Express, and SQLite.

  ## Features
  - CSV upload and data management
  - 7-rule automated matching engine
  - Reconciliation workspace with side-by-side entity panels
  - Dashboard with KPIs and charts
  - Exception management
  - Manual reconciliation integrated in workspace
  - Rule configuration UI
  - Audit trail
  - SQLite database (offline-capable)
